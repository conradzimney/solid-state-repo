Experience

The majority of my technical background comes originally from my interest and passion for web development. My first technical class within the Informatics curriculum at the UW was an Introduction to Client-Side Web Development course. Ever since I took this class, my passion for the subject has never dwindled. Originally learning HTML, CSS and Javascript, I then moved on to learn different Javascript libraries including jQuery, D3.js, Node.js, Express.js, some Angular.js and React.js. To advance my web programming capabilities I moved on to learn some non-javascript languages such as PHP and Google's GoLang. I am always interested in learning new web technologies in order to create the best experiences for users, and design the most efficient API's.

When it comes to Front-End web development, I believe in taking a mobile-first design approach, and always creating experiences with end-users perspectives held as most important. For Back-End web development, efficiency and full API life-cycle management is key. Designing efficient API's and well documented code is vital to any successful project, and managing growth or changes to code is crucial for scalability.

Go Web Server

Built a suggestion service that returns possible matches based on what a user has typed so far using the Go Programming Language. These kinds of services need to be very high-performance, so I have used Go to build a compiled executable, and have implemented a trie data structure to make finding matches very efficient.

Mean Web Server

Created a web server and client for a monetary exchange system (like PayPal). The application provides authentication for users, profile management, and a dashboard for viewing transactions. The application is built using MongoDB, Express.js, Angular.js, and Node.js (MEAN) stack. The application does not allow for the actual exchange of real currency, but it mimics how such a monetary exchange system would function. Users can create and manage their profiles by changing their passwords and personal data, while their session data is persisted throughout.

LAMP Web Server

Created a web server and client using Linux, Apache, MySQL and PHP (LAMP), hosted on Digital Ocean, that shows movie box office revenues from 2014. On the list view, users can search for a given movie by title, and on the detail view, users can view information provided from the Open Movie Database API, from IMBD.

Amazon Gift Card Portal

While I was an Intern on the Advanced Analytics team at West Monroe Partners in Seattle, I had the opportunity to collaborate on a team of 5 to develop a global, enterprise-level gift card purchasing portal for Amazon.com. I utilized Salesforce.com, HTML, CSS, and Javascript to help create multiple web pages for customers to purchase large orders of gift cards, as well as test their eligibility for such purchases.
